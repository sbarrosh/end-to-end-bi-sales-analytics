# End-to-End BI Sales Analytics

## 📊 Overview

This project implements a complete Business Intelligence solution, from data ingestion and transformation to data modeling and dashboard visualization.

The goal is to transform raw sales data into actionable insights to support business decision-making.

## 🧱 Architecture

* Data Sources: CSV datasets (sales, customers, products)
* ETL: Python (Pandas) + SQL transformations
* Data Modeling: Star schema (fact + dimensions)
* Visualization: Power BI dashboards

---

## 📂 Data Model

Star schema design:

* Fact Table: `sales`
* Dimensions:

  * `customers`
  * `products`
  * `date`

This structure enables efficient analytical queries and scalable reporting.

---

## ⚙️ ETL Process

* Data cleaning and transformation using Python (Pandas)
* SQL-based transformations for structured analysis
* Data validation to ensure consistency and quality

---

## 📈 Dashboard & KPIs

The Power BI dashboard includes:

* Revenue
* Profit & Profit Margin
* Sales trends over time
* Top products and regions
* Customer segmentation
